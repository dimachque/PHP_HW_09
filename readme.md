## Описание

+ Создан PHP-файл, выведены таблица истинности, таблицы жесткого и гибкого сравнения при помощи HTML, CSS.
+ Заполнены результаты операции отрицания.

+ Заполнены результаты логической операции XOR.
+ Заполнены результаты жесткого и гибкого сравнений.
+ Сделаны выводы о сравнении в PHP.

## Таблица истинности PHP

Четыре логические операции и их значения: отрицание, логическое **`ИЛИ`**, логическое **`И`**, исключающее **`ИЛИ`**. Например, **`!A`** — это логическое отрицание: если **`А = 0`**, то отрицание **`А`** равно **`1`** **`(!А = 1)`**, и наоборот.

> Результаты всех ячеек в таблице проведены результатом PHP-вычислений.

## Таблица сравнения

Приведена матрица с различными значениями по оси **`Y и X`**.

> Результаты всех ячеек в таблице проведены результатом **`PHP`**-вычислений.

+ Использовано гибкое сравнение.
+ Использовано строгое сравнение.
